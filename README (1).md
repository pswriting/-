# 전자책 작성 프로그램 (CASHMAKER)

AI를 활용한 베스트셀러급 전자책 제작 올인원 솔루션

## 주요 기능

- **주제 적합도 분석**: 5개 항목 0-100점 채점
- **타겟 & 컨셉 설정**: 베스트셀러 스타일 제목 생성
- **목차 자동 설계**: 몰입감 있는 6-7개 챕터 구성
- **인터뷰 기반 본문 작성**: 3500-4500자 챕터별 작성
- **문체 다듬기**: 4가지 스타일 (친근한/전문적/직설적/스토리텔링)
- **마케팅 카피 생성**: 크몽 상품명, 인스타/블로그 홍보 문구

## 사용 방법

### 1. API 키 발급 (무료, 2분)

1. [Google AI Studio](https://aistudio.google.com/apikey) 접속
2. Google 계정으로 로그인
3. **"API 키 만들기"** 클릭
4. 생성된 키 복사

### 2. 프로그램 사용

1. 사이드바에서 API 키 입력
2. 탭 순서대로 진행:
   - 주제 선정 → 타겟 & 컨셉 → 목차 설계 → 본문 작성 → 문체 다듬기 → 최종 출력

## 로컬 실행

```bash
pip install -r requirements.txt
streamlit run ebook_writer.py
```

## 기술 스택

- Python + Streamlit
- Google Gemini API (gemini-2.0-flash)
